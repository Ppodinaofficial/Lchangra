# Lchangra - Real-time Chat & Video Call Platform

A modern web application for real-time group chat and 1:1 video calls using WebRTC.

## Features

- Join rooms with temporary usernames
- Real-time group chat
- See online users in the room
- Start 1:1 video calls with other users
- Modern, responsive UI

## Installation

1. Extract all files to a folder named "Lchangra"
2. Open command prompt in the Lchangra folder
3. Run: `npm install`
4. Run: `npm start`
5. Open http://localhost:3000 in your browser

## How to Use

1. Enter your username and a room name
2. Click "Join Room"
3. Start chatting with others in the room
4. Click the video icon next to any user to start a 1:1 video call
5. Accept/reject incoming video calls

## Technology Stack

- Node.js + Express
- Socket.io for real-time communication
- WebRTC for video calls
- Vanilla JavaScript, HTML5, CSS3
